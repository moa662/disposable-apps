## 📱 项目概述

- ✅ PWA版本已完成（trip.html + manifest.json + sw.js）

### 核心需求
1. **双平台支持**：iOS（iOS 15+）和 Android（API 26+）
2. **离线可用**：本地数据存储，无网络正常运行

4. **快速交付**：基于现有PWA代码迁移

## 🎯 技术方案对比

### 方案一：React Native（推荐 ⭐⭐⭐⭐⭐）

**优势**
- 🎯 与现有PWA技术栈高度兼容（JavaScript/TypeScript）
- ⚡ 开发速度快，热重载支持
- 📦 生态丰富，社区活跃
- 🔄 可直接复用PWA的逻辑代码
- 🔧 原生模块扩展能力强

**劣势**
- ⚠️ 包体积稍大（约8-15MB）
- 🐛 偶现原生桥接问题（可通过优化避免）

**适用场景**
- 快速迭代，需要频繁更新
- 团队有React/前端经验
- 预算有限，追求开发效率

---

### 方案二：Flutter（备选 ⭐⭐⭐⭐）

**优势**
- 🎨 一致性高，iOS/Android渲染一致
- ⚡ 性能卓越（60fps稳定）
- 🔨 丰富的UI组件库
- 🛠️ 强大的Dart语言，类型安全

**劣势**
- ❌ 需要学习Dart语言
- ⏱️ 与PWA代码复用度低
- 📦 包体积较大（约10-20MB）

**适用场景**
- 追求极致性能和一致体验
- 团队愿意学习新技术
- 应用复杂度高，动画要求多

---

### 方案三：原生双开（不推荐 ⭐⭐）

**优势**
- ✅ 最佳性能和用户体验
- 🔧 完全原生能力访问

**劣势**
- 💰 开发成本高（2倍开发工作量）
- ⏰ 维护成本高（两套代码）
- 🚫 与现有PWA代码无法复用

**适用场景**
- 对性能要求极高（游戏、AR/VR）
- 需要深度集成平台特有能力

```typescript
  contacts: Contact[];
  mapUrl?: string;

  name: string;

  wechat?: string;
  email?: string;

export interface TripData {

  startDate: string;
```

### 存储实现

```typescript
// services/storageService.ts
import AsyncStorage from '@react-native-async-storage/async-storage';

const TRIP_DATA_KEY = '@trip_data';

export const storageService = {
  async saveTripData(data: TripData): Promise<void> {
    try {
      await AsyncStorage.setItem(TRIP_DATA_KEY, JSON.stringify(data));
    } catch (error) {
      console.error('保存失败:', error);
    }
  },

  async getTripData(): Promise<TripData | null> {
    try {
      const json = await AsyncStorage.getItem(TRIP_DATA_KEY);
      return json ? JSON.parse(json) : null;
    } catch (error) {
      console.error('读取失败:', error);
      return null;
    }
  },

  async clearData(): Promise<void> {
    await AsyncStorage.removeItem(TRIP_DATA_KEY);
  },
};
```

**功能特性**
- 📍 顶部Hero区域：标题、日期范围、人数
- 🗓️ 日期总览卡片：6天快速跳转

- ⚠️ 待办提醒：红色高亮
- 🔄 滚动联动：浏览时高亮对应日期

**核心代码示例**

```typescript
// screens/TripScreen.tsx
import React, { useState, useEffect, useRef } from 'react';
import {
  View,
  Text,
  ScrollView,
  TouchableOpacity,
  StyleSheet,
  Dimensions,
} from 'react-native';
import { useTripStore } from '../store/tripStore';

const { width } = Dimensions.get('window');

export const TripScreen: React.FC = () => {
  const { tripData, selectedDay, setSelectedDay } = useTripStore();
  const scrollViewRef = useRef<ScrollView>(null);
  const [expandedDays, setExpandedDays] = useState<Set<number>>(new Set([0]));

  // 滚动监听，联动高亮日期
  const handleScroll = (event: any) => {
    const yOffset = event.nativeEvent.contentOffset.y;
    const screenHeight = Dimensions.get('window').height;
    const currentDay = Math.floor(yOffset / (screenHeight * 0.8));

    if (currentDay !== selectedDay && currentDay >= 0 && currentDay < tripData.days.length) {
      setSelectedDay(currentDay);
    }
  };

  const handleDayPress = (dayIndex: number) => {
    setSelectedDay(dayIndex);
    setExpandedDays(prev => new Set([...prev, dayIndex]));

    scrollViewRef.current?.scrollTo({
      y: dayIndex * (Dimensions.get('window').height * 0.8),
      animated: true,
    });
  };

  const toggleDay = (dayIndex: number) => {
    setExpandedDays(prev => {
      const newSet = new Set(prev);
      if (newSet.has(dayIndex)) {
        newSet.delete(dayIndex);
      } else {
        newSet.add(dayIndex);
      }
      return newSet;
    });
  };

  return (
    <View style={styles.container}>
      {/* Hero区域 */}
      <View style={styles.hero}>
        <Text style={styles.title}>{tripData.title}</Text>
        <Text style={styles.subtitle}>
          {tripData.startDate} - {tripData.endDate} · {tripData.people}人
        </Text>
      </View>

      {/* 日期总览 */}
      <ScrollView
        horizontal
        showsHorizontalScrollIndicator={false}
        style={styles.dateOverview}
      >
        {tripData.days.map((_, index) => (
          <TouchableOpacity
            key={index}
            style={[
              styles.dateCard,
              selectedDay === index && styles.dateCardActive,
            ]}
            onPress={() => handleDayPress(index)}
          >
            <Text style={[
              styles.dateText,
              selectedDay === index && styles.dateTextActive,
            ]}>
              第{index + 1}天
            </Text>
          </TouchableOpacity>
        ))}
      </ScrollView>

      {/* 行程列表 */}
      <ScrollView
        ref={scrollViewRef}
        onScroll={handleScroll}
        scrollEventThrottle={400}
        showsVerticalScrollIndicator={false}
        style={styles.tripList}
      >
        {tripData.days.map((dayEvents, dayIndex) => (
          <View key={dayIndex} style={styles.daySection}>
            <TouchableOpacity
              style={styles.dayHeader}
              onPress={() => toggleDay(dayIndex)}
            >
              <Text style={styles.dayTitle}>第{dayIndex + 1}天</Text>
              <Text style={styles.dayExpand}>
                {expandedDays.has(dayIndex) ? '收起' : '展开'}
              </Text>
            </TouchableOpacity>

            {expandedDays.has(dayIndex) && (
              <View style={styles.eventsContainer}>
                {dayEvents.map(event => (
                  <View
                    key={event.id}
                    style={[
                      styles.eventCard,
                      event.isTodo && styles.todoCard,
                    ]}
                  >
                    <Text style={styles.eventTime}>{event.time}</Text>
                    {event.timeHint && (
                      <Text style={styles.timeHint}>{event.timeHint}</Text>
                    )}
                    <Text style={styles.eventTitle}>{event.title}</Text>
                    {event.description && (
                      <Text style={styles.eventDesc}>{event.description}</Text>
                    )}
                  </View>
                ))}
              </View>
            )}
          </View>
        ))}
      </ScrollView>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#0f1117',
  },
  hero: {
    padding: 20,
    paddingTop: 60,
  },
  title: {
    fontSize: 28,
    fontWeight: '700',
    color: '#ffffff',
    marginBottom: 8,
  },
  subtitle: {
    fontSize: 15,
    color: '#a0a0a0',
  },
  dateOverview: {
    paddingHorizontal: 16,
    marginBottom: 16,
  },
  dateCard: {
    paddingHorizontal: 16,
    paddingVertical: 8,
    marginRight: 8,
    borderRadius: 8,
    backgroundColor: '#1a1d27',
  },
  dateCardActive: {
    backgroundColor: '#c9a96e',
  },
  dateText: {
    fontSize: 14,
    color: '#a0a0a0',
  },
  dateTextActive: {
    color: '#0f1117',
    fontWeight: '600',
  },
  tripList: {
    flex: 1,
    paddingHorizontal: 16,
  },
  daySection: {
    marginBottom: 16,
  },
  dayHeader: {
    flexDirection: 'row',
    justifyContent: 'space-between',
    alignItems: 'center',
    marginBottom: 12,
  },
  dayTitle: {
    fontSize: 18,

  },
```

### 3. 状态管理

```typescript
// store/tripStore.ts
import { create } from 'zustand';
import { TripData } from '../types';
import { storageService } from '../services/storageService';

interface TripStore {
  tripData: TripData | null;
  selectedDay: number;
  isLoading: boolean;
  loadTripData: () => Promise<void>;
  updateTripData: (data: Partial<TripData>) => void;
  setSelectedDay: (day: number) => void;
}

export const useTripStore = create<TripStore>((set) => ({
  tripData: null,
  selectedDay: 0,
  isLoading: true,

  loadTripData: async () => {
    set({ isLoading: true });
    const data = await storageService.getTripData();
    set({ tripData: data, isLoading: false });
  },

  updateTripData: (newData) => {
    set((state) => {
      const updatedData = state.tripData
        ? { ...state.tripData, ...newData }
        : (newData as TripData);
      storageService.saveTripData(updatedData);
      return { tripData: updatedData };
    });
  },

  setSelectedDay: (day) => set({ selectedDay: day }),
}));
```
