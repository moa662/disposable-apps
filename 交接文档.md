---

## PWA 功能实现

### 1. manifest.json 配置
- 应用名称：杭州出差行程 / 杭州行
- 启动页：./trip.html
- 显示模式：standalone（独立全屏，无浏览器地址栏）
- 主题色：#0f1117（深色）
- 图标：SVG 内联，支持 192x192 和 512x512

### 2. sw.js 缓存策略
- 缓存文件：trip.html、manifest.json
- 网络优先策略（有网用网，无网用缓存）
- 缓存版本：hangzhou-trip-v1

### 3. HTML PWA 集成
```html
<head>
  <link rel="manifest" href="manifest.json">
  <meta name="theme-color" content="#0f1117">
  <meta name="apple-mobile-web-app-capable" content="yes">
  <meta name="apple-mobile-web-app-title" content="杭州行">
</head>
<script>
  if('serviceWorker' in navigator){
    navigator.serviceWorker.register('./sw.js');
  }
</script>
```

## 页面结构

### 1. 行程页（默认首页）
- 顶部 Hero：标题、日期、人员数
- 总览时间线：6天快速跳转卡片
- 每日行程卡片：可展开/收起，含时间建议
- 待办提醒：红色高亮卡片

### 2. 住宿页（底部导航切换）

- 润宇附近酒店（待定）：红色边框提示
- 交通参考：机场/东站/站到酒店时间

### 3. 联系人页（底部导航切换）
- 海外海大酒店：张经理电话、微信
- 航海家AI大会：地点、签到时间
- 润宇实战课：地点、时间、需带物品

## 设计规范

### 字体大小
- html 基准：17px
- 时间文字：15px，字重 700
- 时间建议：11px，灰色
- 事件标题：16px
- 事件正文：14px
- 待办提醒：14-15px

### 颜色
- 背景：#0f1117
- 卡片：#1a1d27
- 强调色：#c9a96e（金色）
- 危险色：#e85d5d（红色，用于提醒）
- 成功色：#5bc4a0
- 蓝色：#5b8def

### 交互
- 点击日期卡片：滚动到对应天并展开
- 滚动联动：浏览到某天时顶部总览自动高亮
- 电话链接：`tel:` 协议直接拨打
- 地图链接：高德地图 `uri.amap.com`

## GitHub Pages 启用步骤

**目标**：启用后可通过 https://moa662.github.io/disposable-apps/trip.html 访问

1. 打开仓库设置：https://github.com/moa662/disposable-apps/settings/pages
2. **Build and deployment** 部分，**Source** 选择：
   - **Deploy from a branch**
   - Branch: `master`
   - Folder: `/ (root)`
3. 点击 **Save**
4. 等1-2分钟，访问 https://moa662.github.io/disposable-apps/trip.html

**注意**：如果 Save 按钮灰色，可能是权限问题或未修改选项。

## 手机安装 PWA

### 方法一：GitHub Pages（推荐）
1. 手机浏览器打开：https://moa662.github.io/disposable-apps/trip.html
2. Safari/iOS：分享 → "添加到主屏幕"
3. Chrome/Android：菜单 → "添加到主屏幕" / "安装应用"

### 方法二：本地文件
1. 将 trip.html、manifest.json、sw.js 传到手机
2. 浏览器打开 trip.html
3. iOS Safari：分享 → "添加到主屏幕"
