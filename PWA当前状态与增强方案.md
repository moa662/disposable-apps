## 📱 当前使用方式

### 方法1：GitHub Pages（推荐，待启用）

1. **启用步骤**：
   - 打开：https://github.com/moa662/disposable-apps/settings/pages
   - Source → Deploy from a branch → Branch: master → Folder: / (root)
   - 点击 Save

2. **安装到手机**：
   - 手机浏览器打开：https://moa662.github.io/disposable-apps/trip.html
   - iOS Safari：分享 → "添加到主屏幕"
   - Android Chrome：菜单 → "添加到主屏幕" / "安装应用"

**优势**：
- ✅ 一键安装，无需传文件
- ✅ 自动更新（修改代码后刷新即生效）
- ✅ 可通过链接分享给同事

### 方法2：本地文件

1. 将 trip.html、manifest.json、sw.js 传到手机
2. 浏览器打开 trip.html
3. 添加到主屏幕

**限制**：
- ⚠️ file:// 协议下 Service Worker 不运行（但功能可用）
- ⚠️ 需要手动传文件，更新麻烦

## 🚀 PWA增强方案（可选）

如果未来需要增强PWA功能，可以考虑：

### 方案1：数据编辑功能

**功能**：支持在手机上修改行程、住宿、联系人信息

**实现**：
- 使用 localStorage 存储修改内容
- 添加编辑按钮（每张卡片右上角）
- 弹出表单进行编辑
- 修改后立即保存到本地

**预计开发时间**：2-3小时

---

### 方案2：推送通知提醒

**功能**：行程前一天发送提醒通知

**实现**：
- 使用 Web Push API
- 需要 VAPID 密钥和后端服务
- 或使用第三方服务（如 OneSignal）

**预计开发时间**：4-6小时 + 后端服务

---

### 方案3：数据导入/导出

- 导出为 JSON 文件

**预计开发时间**：3-4小时

## 🔧 快速修改指南

如果需要修改内容，直接编辑 trip.html：

### 修改行程数据

找到 `const tripData` 对象，修改对应字段：

```javascript
const tripData = {
  title: "杭州出差行程",
  startDate: "3月27日",
  endDate: "4月1日",
  people: 2,
  days: [
    // 第1天数据
    [
      { time: "20:50", timeHint: "建议提前2小时", title: "长龙航GJ8980", description: "邯郸→萧山T3", type: "flight" },
      // ... 更多事件
    ],
    // 第2-6天数据
  ],
  hotels: [...],
  contacts: [...]
};
```

### 修改颜色/字体

找到 `<style>` 标签，修改 CSS 变量：

```css
:root {
  --bg-color: #0f1117;
  --card-bg: #1a1d27;
  --primary: #c9a96e;
  --danger: #e85d5d;
}
```

### 添加新的事件类型

1. 在 `days` 数组中添加事件
2. 设置 `type` 字段（已有：flight, meeting, free, checkin）
3. 可选：添加新的 type 对应的图标样式
